## Goal:

1. Configure dynamic NAT on R1.
> Configure the appropriate inside/outside interfaces
> Translate all traffic from 172.16.0.0/24
> Create a pool of 100.0.0.1 to 100.0.0.2 from the 100.0.0.0/24 subnet

2. Clear the NAT translations and remove the current NAT configuration.
Switch the configuration to PAT using R1's public IP address

## 1. Configure- dynamic NAT on R1


```
R1(configure)# interface g0/0
R1(configure-if)# ip nat outside
R1(configure)# interface g0/1
R1(configure-if)# ip nat inside

R1(configure)# ip access-list 1 permit 172.16.0.0  0.0.0.255 #creat ACL 
R1(configure)# ip nat pool POOL1 100.0.0.1 100.0.0.2 netmask 255.255.255.0
R1(configure)# ip nat inside source list 1 pool POOL1 
R1# show ip nat translations
```


## Configure-  Clear the NAT translations and configure PAT 

```
R1#clear ip nat transation
R1(configure)do show run | include nat
R1(configure)ip nat inside source list 1 interface g0/0 overload
```
