

## Goal:

1. Configure a default route to the Internet on R1.
2. Configure R1 to use 1.1.1.1 as its DNS server and configure host entries on R1 for R1, PC1, PC2, and PC3.

## Configuration - default route on the internet on R1

```
R1(config)#ip route 0.0.0.0 0.0.0.0 203.0.113.2
R1(config)#do show ip route 
```

Configuration -  R1 to use 1.1.1.1 as its DNS
```
R1(config)#ip name-server 1.1.1.1 
```

Configuration - configure host entries R1/PC1/PC2/PC3
```
R1(config)#ip host PC1 192.168.0.1 
R1(config)#ip host PC2 192.168.0.2
R1(config)#ip host PC3 192.168.0.3
R1(config)#ip host R1 192.168.0.254
R1(config)#do show hosts
```

*Tool : Packer Tracer
* Packer tracer can't support the configuration , such as ip dns server .
