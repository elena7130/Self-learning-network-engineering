## Goal

IPv6 addresses have been pre-configured on the routers.
The serial connections use link-local addresses only

1. Enable IPv6 routing on each router.
2. Configure static routes on the routers to allow PC1 and PC2 to ping each other. The path via R2 should be used only as a backup path.

## Configure -- Enable IPv6 routing on each router.

```
R1(confige)ipv6 unicast-routing
R2(confige)ipv6 unicast-routing
R3(confige)ipv6 unicast-routing
```


## Configure - static routes and  a backup path.

Before configure route , I use `show ipv6 interface brief` to check interface IPv6 address I need and copy them on note .

R1 G0/1 IPV6 address :FE80::202:4AFF:FE23:E202
R1 S0/0/0  IPV6 address :FE80::202:4AFF:FE23:E201
R3 G0/1 IPV6 address :FE80::290:2BFF:FECC:A102
R3 S0/0/0  IPV6 address :FE80::290:2BFF:FECC:A101
R2 S0/0/0 IPV6 address :FE80::20B:BEFF:FED7:4901
R2 S0/0/1  IPV6 address :FE80::20B:BEFF:FED7:4901

```
R1(config)ipv6 route 2001:db8:0:3::/64 g0/1 FE80::290:2BFF:FECC:A102
# From R1 to R3
R1(config)ipv6 route 2001:db8:0:3::/64 S0/0/0 FE80::20B:BEFF:FED7:4901 5
# From R1 to R3 as a backup 

R3(config)ipv6 route 2001:db8:0:1::/64 g0/1 FE80::202:4AFF:FE23:E202
R3(config)ipv6 route 2001:db8:0:1::/64 S0/0/0 FE80::20B:BEFF:FED7:4901 5

R2(config)ipv6 route 2001:db8:0:3::/64 S0/0/1 FE80::290:2BFF:FECC:A101
R2(config)ipv6 route 2001:db8:0:1::/64 S0/0/0 FE80::202:4AFF:FE23:E201

```
