## Goal

Interfaces are enabled and configured with IPv4.
You will configure IPv6 in the network.
1. Use EUI-64 to configure IPv6 addresses on G0/1 of R1/R2
2. Configure the appropriate IPv6 addresses/default gateways on PC1 and PC2.
3. Enable IPv6 on G0/0 of R1/R2 without explicitly configuring an IPv6 address.
4. Configure static routes on R1/R2 to enable PC1 to ping PC2.

## Configure EUI-64 on R1

```
R1(confige)ipv6 unicast-routing
R1(confige)interface g0/1
R1(confige-if)ipv6 2001:db::1/64 eui-64
R1(confige-if)do show ipv6 interface brief
```

```
R2(confige)ipv6 unicast-routing
R2(confige)interface g0/1
R2(confige-if)ipv6 2001:db8:0:1::1/64 eui-64
R2(confige-if)do show ipv6 interface brief
```
## Configure IPV6 addresses and default gateway 

- on PC1 : 
IPV6 address :2001:db8::2 ; 
gateway :2001:db8::1

- on PC2:
IPV6 address :2001:db8:0:1::2 ; 
gateway ::2001:db8:0:1::1

## Configure IP6 address without explicitly configuring on R1 and R2

```
ipv6 enable
```

## Configure IP6 route 

```
R1(confige)do sh ipv6 interface brief
# show the g0/0 ipv6 on R1 address is FE80::230:F2FF:FE36:4501 (link local address)

R2(confige)do sh ipv6 interface brief
# show the g0/0 ipv6 on R2 address is FE80::230:F2FF:FE36:4501 (link local address)

R1(confige)ipv6 route 2001:db8:0:1::/64   g0/0 FE80::230:F2FF:FE36:4501
R2(confige)ipv6 route 2001:db8::/64  g0/0 FE80::230:F2FF:FE36:4501

```
