
## Goal:

1. Configure static NAT on R1.
    Configure the appropriate inside/outside interfaces
    Map the IP addresses of PC1, PC2, and PC3 to 100.0.0.x/24
    
2. Ping google.com from each PC, and then check the NAT translations on R1.
3. Clear the NAT translations on R1. Which entries remain?

## Configure-  static NAT on R1

	* Before I configure the Nat on R1, the PC1 can't ping 8.8.8.8
```
R1(configure)# interface g0/0
R1(configure-if)# ip nat outside
R1(configure)# interface g0/1
R1(configure-if-if)# ip nat inside
R1(configure)# ip nat inside source static 172.16.0.1 100.0.0.1
R1(configure)# ip nat inside source static 172.16.0.2 100.0.0.2
R1(configure)# ip nat inside source static 172.16.0.3 100.0.0.3
R1# show ip nat translations
```
## Configure-  Clear the NAT translations 

```
R1#clear ip nat transation
```
