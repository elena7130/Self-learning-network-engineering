
## Goal:

1. Configure the DHCP pools on R2

- POOL1: 192.168.1.0/24 (reserve .1 to .10)
  DNS 8.8.8.8
  Domain: hiaitools.com
  Default Gateway: R1
- POOL2: 192.168.2.0/24 (reserve .1 to .10)
  DNS 8.8.8.8
  Domain: hiaitools.com
  Default Gateway: R2
- POOL3: 203.0.113.0/30 (reserve .1)

2. Configure the R1's interface as a DHCP client 
3. Configure R1 as a DHCP relay agent for the 192.168.1.0/24 subnet.
4. Use the CLI of PC1 and PC2 to make them request an IP address

## Configuration - DHCP pools on R2(DHCP server)

```
R2(config)#ip dhcp exclued-address 192.168.1.1 192.168.1.10
R2(config)#ip dhcp exclued-address 192.168.2.1 192.168.2.10
R2(config)#ip dhcp exclued-address 203.0.113.1
R2(config)#ip dhcp pool pool1
R2(config)#network 192.168.1.0 255.255.255.0
R2(config)#dns-sesrver 8.8.8.8
R2(config)#domain-name hiaitools.com
R2(config)#defalut-router 192.168.1.1
# set pool1

R2(config)#ip dhcp pool pool2
R2(config)#network 192.168.2.0 255.255.255.0
R2(config)#dns-sesrver 8.8.8.8
R2(config)#domain-name hiaitools.com
R2(config)#defalut-router 192.168.2.1
# set pool2

R2(config)#ip dhcp pool pool3
R2(config)#network 203.0.113.0 255.255.255.252

# set pool3
```

## Configuration - DHCP pools on R1(DHCP client)

```
R1(config)#interface g0/0
R1(config)#ip address dhcp 
R1(config)#no shutdown
```

## Configuration - DHCP pools on R1(DHCP relay agent)

```
R1(config)#interface g0/1
R1(config)#ip help-address 203.0.113.1
```
