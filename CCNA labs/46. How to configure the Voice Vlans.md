
## Goal:

1. Configure SW1's interfaces in the appropriate VLANs.
2. Configure ROAS for the connection between SW1 and R1.

## Configuration - SW1

```
sw1(config)int range g1/0/2 -3 
sw1(config-if-range)swtichport mode access
sw1(config-if-range)swtichport acc vlan 10 
sw1(config-if-range)swtichport voice vlan 20
sw1(config)int range g1/0/1
sw1(config-if)swtichport trunk en dot1q
sw1(config-if)switch mode trunk allowed vlan 10,20
```

## Configuration - R1 

```
R1(config)int f0/0.10
R1(config-if)encapsulation do1q 10 
R1(config-if)ip add 192.168.10.1 255.255.255.0
R1(config)int f0/0.20
R1(config-if)encapsulation do1q 20 
R1(config-if)ip add 192.168.20.1 255.255.255.0
```
