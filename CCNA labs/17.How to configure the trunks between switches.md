## Goal

  1. Configure the switch interfaces connected to PCs as access ports in the correct VLAN.
  2. Configure the connection between SW1 and SW2 as a trunk, allowing only the necessary VLANs. Configure an unused VLAN as the native VLAN.
  3. Configure the connection between SW2 and R1 using 'router on a stick'.

### **Configuration** -- Configure the access ports

```
SW1(config)interface range f0/1-2
SW1(config-if-range)switch mode access
SW1(config-if-range)switch access vlan 10 
SW1(config-if-range)interface range f0/3-4
SW1(config-if-range)switch mode access
SW1(config-if-range)switch access vlan 30
------
SW2(config)interface f0/1
SW2(config-if)switch mode access
SW2(config-if)switch access vlan 20
SW2(config)interface range f0/2-3
SW2(config-if-range)switch mode access
SW2(config-if-range)switch access vlan 10
SW1(config-if-range)do show int trunk
```


### **Configuration** -- Configure trunks and native trunk

```
SW1(config)interface g0/1
SW1(config-if)switch mode trunk
SW1(config-if)switch trunk allowed vlan 10,30
SW1(config-if)switch trunk native vlan 1001


SW2(config)interface g0/1
SW2(config-if)switch mode trunk
SW2(config-if)switch trunk allowed vlan 10,30
SW2(config-if)switch trunk native vlan 1001

SW2(config)interface g0/2
SW2(config-if)switch mode trunk
SW2(config-if)switch trunk allowed vlan 10,20,30
SW2(config-if)switch trunk native vlan 1001

```
### **Configuration** -- Configure router using 'router on a stick'.

```
R1(Config)int g0/0
R1(Config-if)no shutdown
R1(Config)int g0/0.10
R1(Config)encapsulation dot1Q 10
R1(Config-if)ip add 10.0.0.62 255.255.255.192
R1(Config)int g0/0.20
R1(Config)encapsulation dot1Q 20
R1(Config-if)ip add 10.0.0.126 255.255.255.192
R1(Config)int g0/0.30
R1(Config)encapsulation dot1Q 30
R1(Config-if)ip add 10.0.0.190 255.255.255.192
```
